This package contains EVERYTHING needed to build your C.A.R.E.N. AAB file.

═══════════════════════════════════════════════════════════════
STEP 1: UPLOAD FILES TO GITHUB
═══════════════════════════════════════════════════════════════

1. Go to: https://github.com/LordEnki7/caren-app
2. Click "Add file" → "Upload files"
3. Drag ALL FILES from this folder into GitHub
4. Commit message: "Complete AAB build system"
5. Click "Commit changes"

═══════════════════════════════════════════════════════════════
STEP 2: CREATE KEYSTORE & ADD SECRETS
═══════════════════════════════════════════════════════════════

ON YOUR COMPUTER, open Terminal/Command Prompt:

Windows:
certutil -encode keystore.jks temp.txt && type temp.txt

Mac/Linux:
base64 -i keystore.jks

Then go to: https://github.com/LordEnki7/caren-app/settings/secrets/actions

Add these 3 secrets:
- KEYSTORE_BASE64: (paste the base64 output)
- KEYSTORE_PASSWORD: carenstore123
- KEY_PASSWORD: carenstore123

═══════════════════════════════════════════════════════════════
STEP 3: BUILD YOUR AAB
═══════════════════════════════════════════════════════════════

1. Go to: https://github.com/LordEnki7/caren-app/actions
2. Click "Build Android AAB for Google Play Store"
3. Click "Run workflow"
4. Wait 5-10 minutes
5. Download your AAB file from "Artifacts"

═══════════════════════════════════════════════════════════════
ALTERNATIVE: USE ANDROID STUDIO (SIMPLER!)
═══════════════════════════════════════════════════════════════

If GitHub build fails, you can build locally with Android Studio:

1. Extract this folder to your computer
2. Open Terminal in this folder
3. Run: npm install
4. Run: npm run build
5. Run: npx cap sync android
6. Open Android Studio
7. Open the "android" folder
8. Build → Generate Signed Bundle / APK
9. Follow the wizard to create your AAB

═══════════════════════════════════════════════════════════════

and secrets are configured.
